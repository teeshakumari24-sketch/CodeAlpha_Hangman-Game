# CodeAlpha_Hangman-Game
## Overview

The Hangman Game is a simple text-based Python application developed as part of the CodeAlpha Python Programming Internship. The project demonstrates the use of Python fundamentals such as loops, conditional statements, strings, lists, user input, and basic game logic.

In this game, the computer randomly selects a word from a predefined list, and the player attempts to guess the word one letter at a time. For every incorrect guess, the player loses one chance. The game continues until the player either correctly guesses the complete word or reaches the maximum number of incorrect attempts.

The project was created to strengthen problem-solving skills and improve understanding of Python programming through a practical implementation of game logic.

## Features

- Random word selection
- Letter-by-letter guessing
- Input validation
- Tracks guessed letters
- Displays current word progress
- Limits incorrect guesses
- Win and lose conditions
- Beginner-friendly console interface

## Technologies Used

- Python 3
- Random Module
